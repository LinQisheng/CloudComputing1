# Gestor de Tareas (TaskManager)
Aplicación web desarrollada en **PHP, MySQL y JavaScript**, con interfaz simple y API RESTful.

## Funcionalidades
- Registro e inicio de sesión de usuarios
- Creación y gestión de proyectos
- Asignación de tareas y comentarios
- Persistencia en MySQL
- Compatible con XAMPP y cPanel

## Instalación local
1. Clona este repositorio:
   ```bash
   git clone https://github.com/linqisheng/taskmanager-web.git
2. Copia la carpeta en htdocs y crea la base de datos con api/schema.sql

3. Abre en el navegador: http://localhost/taskmanager
